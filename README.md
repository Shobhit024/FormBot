
FormBot
FormBot is an intuitive platform that allows users to create custom forms with multiple options to suit their needs. The forms are dynamic, easy to build, and can be shared freely with others. Whether you're gathering feedback, creating surveys, or collecting user data, FormBot provides an effortless solution to all your form-building needs.

Tech Stack:-
-->Frontend: React.js, Vite
-->Backend: Node.js, Express.js
-->Database: MongoDB
-->Deployment:
-->Backend: Render
-->Frontend: Vercel

Features:-

*Dynamic Form Creation: Users can easily create custom forms by selecting various form elements.
*Live Preview: See real-time changes as you create the form.
*Shareable Forms: Once created, forms can be shared with anyone via a unique link.
*User Authentication (Future Feature): User login for saving and managing forms (coming soon).
*Fully Responsive: Works perfectly on both desktop and mobile devices.

--Deployment
The application is deployed on Render for the backend and Vercel for the frontend. Below are the links to the live application:
